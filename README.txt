Add Shuftipro identity verification process to WordPress websites.

== Description ==

Automatically verify users with ease from anywhere in the world in real-time. From ID document verification to face verification, you can easily perform advanced ID authentication checks to ensure your users are who they claim to be. Add the Shufti KYC plugin to ensure your website is safe from imposters and comply with stringent KYC and AML compliance rules.

Shufti KYC plugin connects your wordpress site with the Shufti Pro’s configurable real-time Identity verification and AML screening platform where you can verify your user identity using different IDV checks. To use this for user verification, you need to activate your subscription with Shufti KYC. You can also use it right away with a [trial account](https://shuftipro.com/try-now/) by getting the trial key from Shufti Pro. Once you are ready to integrate, our experts will facilitate the efficient configuration of Shufti KYC that best fits your needs.

= HOW IT WORTKS =

* Install the Shufti KYC plugin and add it to your wordpress site.
* Get the active subscription of the plugin or start trial mode.
* Enter your client ID and secret key to activate your account.
* Select general parameters against which you want your users to perform verification.
* If you agree with the settings we'll provide you a short-code and save your settings.
* Copy the shortcode and paste it anywhere on the site to start verification.
* Users perform verification without leaving the website.
* Shufti KYC verifies the user in real-time and stores the verification result in compliance with data protection regulation.
* Verification results are stored in the admin dashboard (Shufti Pro’s [back-office](https://backoffice.shuftipro.com/reports)) with proper image and video proof.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `shuftipro-kyc.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enter your `Client id` and `Secret Key`.
4. Select general parameters.
3. Place provided `shortcode` in anywhere on your site.
